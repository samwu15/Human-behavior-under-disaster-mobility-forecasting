# Human-behavior-under-disaster-mobility-forecasting
Capstone project for mobility demand forecasting during natural disasters using spatiotemporal machine learning models.
## Project Overview

This project aims to forecast mobility demand during natural disasters using spatiotemporal machine learning models.

The study uses GPS-based mobility data combined with weather, socioeconomic, and disaster indicators to predict hourly travel demand at the Census Tract level.

The goal is to understand how mobility patterns change during disasters and how forecasting models can adapt to disruption and recovery phases.
